# 万里书院 - 在线作业系统 (Wanli Academy Online Homework System)

本项目为万里书院提供在线作业提交、批阅及视频讲解的核心业务系统。

## 技术架构

- **前端**: Vue.js 3 + TypeScript
- **后端**: Java (Spring Boot 3)
- **数据库**: PostgreSQL
- **部署环境**: Fly.io
- **CI/CD**: GitHub Actions
